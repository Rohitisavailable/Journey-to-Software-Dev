# Journey to Software Dev

A personal learning repository documenting my journey into software development, covering web development fundamentals and Python programming.

## About

This repository contains code, projects, and exercises from my software development learning path. It includes hands-on practice with HTML, CSS, and Python as I work towards becoming a proficient developer.

## Repository Structure

```
├── Web Dev/           # Web development learning modules
│   ├── HTML basics (Headings, Paragraphs, Lists, etc.)
│   ├── CSS fundamentals (Selectors, Colors, Box Model, etc.)
│   └── Projects (Movie Ranking, Birthday Invite, Portfolio, etc.)
├── Python/            # Python programming exercises
├── index.html         # Personal resume webpage
├── Boilerplate.html   # HTML boilerplate template with comments
├── box.html           # CSS box model practice
└── style.css          # CSS styles
```
